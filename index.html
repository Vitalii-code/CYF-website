Open-Source Software

Open-Source Software or OSS for short is a type of software that allows the user to view, study and redistribute the code to other people.

    I like OSS for a few reasons:

  1. It allows anyone to study and learn from the code.
  2. It allows you to contribute to the code if there is something you don't like about a program.
  3. Anyone can audit the program to know that it's safe and secure.

    Some of the disadvantages include:

  1. Sometimes projects "die" because the maintainer loses interest in supporting it(because it's ususally volunteer work).
  2. Sometimes there are limited features because again it's made by volunteers

An example of a few OSS programs featuring Firefox(web browser), LibreOffice (office suite) , VLC(multimedia player) and Thunar(a file manager)

A few more examples of OSS software:

The biggest OSS project would probably be the linux kernel which contains about 40 millions lines of code! And is powering 90% of servers around the world.
